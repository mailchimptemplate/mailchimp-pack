Frequently Asked Questions

Does BumpJar work with standard WooCommerce checkout pages?

Yes! BumpJar works natively with standard WooCommerce checkout shortcodes (`[woocommerce_checkout]`) and virtually all popular WooCommerce themes like Astra, WoodMart, Divi, and OceanWP.

How does the Smart Cross-Sells auto-detection work?

When enabled, BumpJar inspects the products inside the customer's cart. If those products have Linked Cross-Sell items configured in WooCommerce, BumpJar automatically displays that related item as the order bump offer.

Will the audio sound effect slow down my site?

Not at all. The audio chime is generated instantly using modern Web Audio API built right into web browsers. No extra sound files are downloaded, making it 100% lightweight.

Can I offer custom discounts on bump products?

Yes! You can set percentage discounts (e.g. 20% OFF) or fixed amount discounts (e.g. $5 OFF). BumpJar automatically calculates the discount and updates the WooCommerce cart live via AJAX.
