Frequently Asked Questions

How does the Two-Way Sync feature work?

When you edit any product detail (like price, SKU, or stock) in Google Sheets, the spreadsheet fires a secure webhook request using Google Apps Script to our WordPress REST API endpoint, which updates WooCommerce immediately. It is fully encrypted with a Security Token [4].

Will Two-Way Sync create an infinite loop?

No. We have engineered a smart "Bypass Loop Protection" flag. When Google Sheets updates WooCommerce, the plugin recognizes the webhook action and temporarily bypasses the outbound sync, effectively breaking any infinite sync loops [4].

Can I use custom columns in my Google Sheet?

Yes! Our "Dynamic Column Mapping" feature allows you to choose exactly which column letter (A to Z) represents which WooCommerce field. The plugin dynamically structures and pads the array to match your sheet's layout perfectly [4].

Will the background sync lag my WooCommerce checkout page?

Not at all. If you select "Scheduled Sync" mode, new orders are queued as "Pending" in the background [4]. The checkout completes in milliseconds, and our WP Cron job silently processes the queue every 5, 15, or 60 minutes in the background [4].
