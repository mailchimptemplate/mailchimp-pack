Introduction

Chimpjar Contact Chat Builder Pro is a conversion-focused WhatsApp widget for WordPress. It doesn't just show a button; it captures qualified leads by requiring users to fill out a custom form before they start a conversation on WhatsApp.

No API Required

Works with standard WhatsApp or WhatsApp Business accounts without any monthly API costs.

Lead Database

Every click is a lead. Save names, emails, and phone numbers directly into your own WordPress database.

Installation Guide

1

Upload the Plugin

Download the chimpjar-contact-chat-builder-pro.zip from your dashboard. In your WordPress admin, navigate to Plugins > Add New > Upload and choose the file.

2

Activation

After uploading, click Activate. Upon activation, the plugin will automatically create a secure wawb_leads table in your database to store customer data.

Global Appearance Settings

Located at Chimpjar Chat Builder > Settings > Global Settings.

Theme Color & Position

Pick your brand color and set the widget to appear on the bottom-right or bottom-left of the screen.

Pulse Effect

Enable the animated pulse to draw attention to the chat icon. Highly recommended for higher conversion.

Live Preview

The panel on the right allows you to see every change (color, text, size) in real-time before you hit the save button.

Agent Management

This is where you build your support team. You can add unlimited agents for different departments.

WhatsApp Number Format

Always use the international format without + or 00.
CORRECT: 12125550123 | INCORRECT: +1 212...

Profile & Form

Add agent names, designations, and upload avatars. Enable "Lead Capture Form" for specific agents to force a form popup before redirecting to WhatsApp.

Individual Styling

Customize the background, text, and border radius of each agent's card. You can even set unique WhatsApp icon colors for different agents.

Business Hours (Scheduling)

Professional Scheduling

Never get disturbed during your rest time. Each agent can have an independent schedule based on your WordPress site's timezone.

  • Set Start & End Time (24h format)
  • Multi-select Off-Days (e.g. Sat & Sun)
  • Display custom "Offline Text"
  • Option to hide agents when offline

Lead Management & CSV

All captured leads are stored under Chimpjar Chat Builder > Leads.

Smart Triggers & GDPR

Increase engagement by showing the chat widget at the exact right moment using these advanced triggers.

Time Delay

Show widget after X seconds of page load.

Scroll Depth

Show widget after the user scrolls X% of the page.

Exit Intent

Pop-up the widget when the user tries to leave the site.

GDPR Compliance

Enable the GDPR Consent Checkbox in the **Triggers & Lead** tab to stay legal. Users must agree to your privacy policy before starting the chat.

Display Rules & Targeting

Device Visibility

Show or Hide the widget specifically for Desktop or Mobile users to optimize UX.

Page Targeting

Use "Show Only" or "Hide On" rules with page slugs (e.g. /contact, /checkout) to control widget visibility.

💡 Pro Tip:

Hide the widget on checkout pages to avoid distracting customers during the final payment steps.

Shortcode Mastery

Full Floating Widget

[whatsapp_widget]
Main Widget

Automatically loads based on display rules. Use this for manual placement via page builders.

Single Agent (Inline)

[whatsapp id="123"]
Single Agent

Replace 123 with the unique Agent ID found next to the agent's name in settings.
