Getting Started

Welcome to ChimpJar FlipBook Pro! This cloud-based software allows you to build interactive catalogs, presentations, and lead-generating e-books in minutes without writing a single line of code.

Cloud Hosted: You don't need to install anything on your computer. Just log in from your browser, create your flipbook, and embed it anywhere!

How to Redeem AppSumo Code

If you purchased FlipBook Pro from AppSumo, follow these steps to unlock your lifetime PRO features:

  1. Create a free account on our homepage and log in to your Dashboard.
  2. On the left sidebar of the Dashboard, click on the Redeem AppSumo button.
  3. A popup will appear. Paste your unique AppSumo code (e.g., SUMO-ABCD-1234) and click "Redeem Now".
  4. Your account will instantly be upgraded to PRO Lifetime, unlocking Shoppable Hotspots, Custom Colors, Watermarks, Media Embeds, and LeadGen Forms!

Using The Visual Editor

Creating an interactive flipbook is incredibly easy.

Step 1: Load your PDF

  • From the Dashboard, click "Create New Book".
  • In the top bar, paste a valid link to your PDF file (e.g., https://yoursite.com/catalog.pdf) and click Load PDF.
  • Tip: Hosting your PDF on your own server or Google Drive/AWS makes loading lightning fast!

Step 2: Add Interactions (Pro Feature)

  • Shoppable Hotspots: Select the Hotspot tool, click anywhere on the page, and add your product details (Images, Price, Checkout Link) on the right panel.
  • Media Embeds: Select the Media Box tool, click and drag to draw a box on the page, and paste your YouTube, Vimeo, Audio MP3s or Google Maps link.
  • Moving & Resizing: Click and drag a hotspot to move it. Click a media box and drag its yellow handle to resize it.

Step 3: Global Settings & Publishing

  • Branding: Go to the Global Settings tab and change the background, buttons, and hotspot icon colors to match your brand.
  • Watermark: Add a custom diagonal watermark text to protect your document.
  • Publish: Click the green Publish Book button at the top right to save everything. Go back to your Dashboard to grab the Iframe Embed Code!

Google Sheet LeadGen Setup (Pro)

Our powerful Lead Generation feature locks your flipbook on a specific page and asks readers for their email. Follow these steps to save those emails directly to your Google Sheet:

1 Create a Google Sheet

  1. Go to Google Sheets and create a new blank spreadsheet.
  2. Name the columns in the very first row exactly like this: Date, Name, Email, book_id.

2 Add the Apps Script

  1. In your Google Sheet menu, click on Extensions > Apps Script.
  2. Delete any existing code in the editor and paste the code below:
var sheetName = 'Sheet1'
var scriptProp = PropertiesService.getScriptProperties()

function intialSetup () {
  var activeSpreadsheet = SpreadsheetApp.getActiveSpreadsheet()
  scriptProp.setProperty('key', activeSpreadsheet.getId())
}

function doPost (e) {
  var lock = LockService.getScriptLock()
  lock.tryLock(10000)

  try {
    var doc = SpreadsheetApp.openById(scriptProp.getProperty('key'))
    var sheet = doc.getSheetByName(sheetName)

    var headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0]
    var nextRow = sheet.getLastRow() + 1

    var newRow = headers.map(function(header) {
      if(header === 'Date'){ return new Date() }
      return e.parameter[header]
    })

    sheet.getRange(nextRow, 1, 1, newRow.length).setValues([newRow])

    return ContentService
      .createTextOutput(JSON.stringify({ 'result': 'success', 'row': nextRow }))
      .setMimeType(ContentService.MimeType.JSON)
  }

  catch (e) {
    return ContentService
      .createTextOutput(JSON.stringify({ 'result': 'error', 'error': e }))
      .setMimeType(ContentService.MimeType.JSON)
  }

  finally {
    lock.releaseLock()
  }
}

3 Deploy as Web App

  1. Click the blue Deploy button (top right) > New Deployment.
  2. Click the gear icon next to "Select type" and choose Web App.
  3. Fill in the details:
    • Description: FlipBook LeadGen
    • Execute as: Me (your email)
    • Who has access: Anyone (This is very important!)
  4. Click Deploy. You may need to grant permissions using your Google account.
  5. Copy the "Web App URL" provided.

4 Connect to FlipBook

  1. Go to the FlipBook Editor > Global Settings tab > Lead Generation section.
  2. Check the "Enable Form Lock" box.
  3. Set the Page Number (e.g., 4) where the book should lock.
  4. Paste the copied Google Apps Script URL into the input box and click Update Book.
That's it! When readers reach that specific page, they will see a beautiful lock screen. Once they submit the form, their details will instantly appear in your Google Sheet!
